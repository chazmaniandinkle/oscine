# local/

Your machine-local working files: Oscine song exports (`.oscine.json`), local
config, sketches, and anything you are experimenting with.

Nothing in this folder is tracked by git except this README (see the
`/local/*` rule in `.gitignore`), so your work here never shows up in
`git status` as something to commit and never goes upstream by accident.

Save song exports and local config here. Each machine keeps its own `local/`.
