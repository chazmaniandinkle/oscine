# Generated plugin mirror and versioning

- `plugin/app/` is generated by `tools/sync-plugin.mjs`. Never edit files
  under `plugin/app/` directly.
- After any change to `src/`, `index.html`, or `styles/`, run
  `node tools/sync-plugin.mjs`. The smoke suite fails if the bundled copy
  has drifted from the source.
- Changes to the sidecar or plugin should bump the version in
  `plugin/.claude-plugin/plugin.json` and `SERVER_VERSION` in
  `plugin/server/oscine-mcp.mjs`, then resync and repackage the
  `.plugin` file.
